WebGrid is a fast and concise ASP.NET Control that simplifies ASP.NET
development, event handling, data-binding, and Ajax interactions for rapid web development.

Create fast loading web application with a rich client-side of properties and tons of features with a performance that beats ASP.NET GridView.
